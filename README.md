# Micro service node api for Gov Election

### How to use ?

##### Clone the project

##### install dependencies

```

npm install
```

##### Development mode


##### run the project
```
npm start
```

##### Production mode

```
npm run build
npm run start-prod
```

Use de `.env` file to store your important information such as your server port, your password,
